# Bike Sales Excel Dashboard

This project is an interactive **Bike Sales Dashboard built in Microsoft Excel** as part of my data analytics learning journey.

## Objective
To analyze customer purchasing behaviour and identify patterns across demographics and regions.

## Key Metrics
- Total Customers: 1,000
- Total Bike Purchasers: 481
- Purchase Conversion Rate: 48.1%

## Analysis Breakdown
- Purchasing behaviour by:
  - Age group
  - Gender
  - Income range
  - Education level
  - Region
- Interactive slicers for dynamic filtering
- KPI cards for quick insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Calculated Fields
- Slicers
- Data Visualization Techniques

## Key Insight
Middle-aged customers show the highest purchase conversion rate across multiple income categories.
